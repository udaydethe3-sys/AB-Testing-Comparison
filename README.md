# A/B Testing Comparison 2024



## Description
A/B testing analysis comparing two versions of a webpage using Excel.

## Tools Used
- Microsoft Excel
- IF Formulas
- Bar Charts (Conversion Rate & Count)

## What This Project Contains
- 12 weeks of A/B test data
- Version A vs Version B visitor and conversion tracking
- Automatic conversion rate calculation
- Winner detection using IF formula
- Summary dashboard with final recommendation

## Key Findings
- Version B outperformed Version A in all 12 weeks
- Version B had higher conversion rate consistently
- Recommended to deploy Version B as final design
